# Quiz-para-Programadores-
Este é um projeto de quiz desenvolvido com HTML e CSS, criado com o objetivo de permitir que programadores testem seus conhecimentos de forma simples, rápida e interativa.

A ideia do projeto é proporcionar uma experiência leve e intuitiva, com perguntas voltadas para HTML, conceitos básicos e curiosidades da área.

 🚀 Tecnologias utilizadas
- HTML5
- CSS3

🎯 Objetivo do projeto

Este projeto foi desenvolvido para praticar estruturação de páginas com HTML e estilização com CSS, além de treinar organização de código e versionamento com Git e GitHub.

🧩 Funcionalidades

- Perguntas de múltipla escolha
- Layout simples e responsivo
- Feedback visual para respostas

📚 Aprendizados

Durante o desenvolvimento deste projeto, pude praticar:
- Estrutura com HTML
- Estilização com CSS
- Organização de arquivos
- Publicação de projeto no GitHub

🔗 Como visualizar

Basta abrir o arquivo `index.html` no navegador.
